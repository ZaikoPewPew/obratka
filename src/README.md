# `src/` — код приложения

Папка содержит всю клиентскую логику лендинга: сборку интерфейса, локализацию, работу с Supabase и утилиты.

## Файлы верхнего уровня

- `main.js` — точка входа: инициализация UI, событий и отправки email.
- `i18n.js` — загрузка локалей/контента из JSON, выбор текущего языка, доступ к строкам.
- `config.js` — общие константы приложения (например, отображаемое число для счётчика).

## Подпапки

- `components/` — UI-компоненты (создают DOM-узлы и поведение блоков интерфейса).
- `utils/` — мелкие переиспользуемые функции (валидация, форматирование, эффекты).
- `api/` — HTTP-слой для работы с Supabase.
- `assets/` — статические ассеты, импортируемые из кода.
