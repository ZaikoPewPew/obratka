# `src/` — код приложения

Папка содержит клиентскую логику: экраны, локализацию, API и утилиты.

Архитектура экранов: [`SCREENS.md`](../SCREENS.md).

## Файлы верхнего уровня

- `main.js` — точка входа: сейчас iframe-сессия + `url-screen`; флоу app-экранов подключится позже.
- `i18n.js` — загрузка локалей/контента из JSON, выбор текущего языка, доступ к строкам.
- `config.js` — общие константы приложения.

## Подпапки

- `app/` — флоу экранов и сессия пользователя (каркас).
- `components/` — UI-компоненты (создают DOM-узлы и поведение).
- `utils/` — мелкие переиспользуемые функции.
- `api/` — HTTP-слой (Supabase и будущие auth/portfolios).
- `assets/` — статические ассеты, импортируемые из кода.
