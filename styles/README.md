# `styles/` — слои стилей

Папка организована по принципу «от базы к частным сценариям», чтобы проще поддерживать адаптив.

## Состав

- `tokens.css` — единый источник дизайн-токенов: цвета, поверхности, статусы, тени, скругления, отступы, шрифт, брейкпоинт.
- `base.css` — базовый слой (сброс, общие контейнеры, фундаментальные правила).
- `entrance.css` — стили стартового/входного состояния интерфейса.
- `desktop.css` — правила десктопного лейаута и его компонентов.
- `mobile.css` — переопределения и мобильный лейаут (`max-width: 767px`).
- `apply.css` — стили карточки заявки/формы.
- `access-modal.css` — стили модального окна доступа.
- `privacy-policy-panel.css` — стили панели политики конфиденциальности.
- `notification.css` — стили уведомлений.
