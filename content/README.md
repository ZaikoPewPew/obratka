# `content/` — контент и данные

Эта папка хранит контентные JSON-файлы, которые читает `src/i18n.js`.

## Состав

- `locales.json` — тексты интерфейса по языкам, список поддерживаемых локалей, параметры таймера.
- `privacy-policy.json` — HTML-тексты политики конфиденциальности по локалям.
- `founder-avatars.json` — источники аватаров основателей и параметры выборки.

## Почему нет комментариев внутри JSON

Стандарт JSON не поддерживает комментарии (`//` и `/* */` недопустимы), поэтому пояснения вынесены в этот `README.md`, чтобы не ломать парсинг в приложении.
