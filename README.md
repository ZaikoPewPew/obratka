# Memento — waitlist

Статический лендинг вейтлиста: **Vite**, **vanilla JS**, анимация падающих карточек стартапов на **Matter.js**, локализация через `content/locales.json`.

## Быстрый старт

Требуется Node.js (рекомендуется актуальная LTS).

```bash
npm install
npm run dev
```

Откройте в браузере адрес, который выведет Vite (обычно `http://localhost:5173`).

## Скрипты

| Команда | Назначение |
|---------|------------|
| `npm run dev` | Режим разработки с hot reload |
| `npm run build` | Сборка в каталог `dist/` |
| `npm run preview` | Локальный просмотр production-сборки |

## Документация в репозитории

- [`PROJECT.md`](PROJECT.md) — брейкпоинты, два лейаута (десктоп / мобилка), структура файлов.
- [`mobile.md`](mobile.md) — спецификация мобильного макета (отступы, типографика, размеры формы и фона).
